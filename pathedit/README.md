# EzVibeCode (EVC) — Быстрый старт

Привет! `EVC` — это легкая консольная утилита, которая собирает файлы твоего проекта в один текстовый файл, чтобы ты мог быстро скормить его любой нейросети (ChatGPT, Claude, Gemini, Qwen и не дай бог DeepSeek и др.) как полный контекст.

## Что внутри архива?
* `EVC.exe` — сам исполняемый файл утилиты.
* `config.json` — настройки (какие расширения читать, какие папки игнорировать).
* `pathedit.exe` — мини-утилита, чтобы тебе не пришлось прописывать пути вручную.
* `README.md` — эта инструкция.

---

## Установка за 3 простых шага

### Шаг 1. Распаковка
Создай в удобном месте на диске (например, на диске `C:`) папку для утилит. Рекомендуемый путь:
`C:\tools\EVC\`

Распакуй **все четыре файла** из этого архива прямо в эту созданную папку.

### Шаг 2. Запуск авто-настройки
1. Открой папку `C:\tools\EVC\` и запусти файл **`pathedit.exe`** (дважды кликни на него).
2. Программа автоматически предложит сделать текущую папку глобальной. Просто **нажми Enter** для подтверждения.
3. На вопрос `Подтверждаете добавление этой папки в PATH?` введи `y` (или `д`) и **нажми Enter**.
4. Всё готово! Закрой окно программы и после настройки PATH утилита больше не понадобится.

### Шаг 3. Перезапуск терминала
Если у тебя был открыт VS Code, обычная консоль PowerShell или CMD — **закрой их и открой заново**, чтобы система обновила переменные окружения.

---
### Если программа не запускается

- Windows Defender мог поместить файл в карантин
- Запустите от имени администратора
- Убедитесь, что config.json лежит рядом с EVC.exe

---

## Как пользоваться

Теперь ты можешь вызвать `EVC` из абсолютно **любого места на компьютере**.

1. Открой терминал/консоль в папке со своим проектом, который хочешь отправить в нейросеть.
2. Просто напиши:
   ```powershell
   EVC
   ```
3. Программа прочитает настройки из `C:\tools\EVC\config.json` и создаст готовый файл контекста прямо в папке твоего проекта.

### Полезные фичи и флаги запуска:
Вы можете переопределять поведение утилиты на лету:
* `EVC -t tests` — запустить сбор контекста, проигнорировав дополнительную папку `tests`.
* `EVC -nd` — собрать контекст без вывода дерева проекта в конце результирующего файла.

## Тонкая настройка (`config.json`)
Если нужно добавить новые форматы файлов для сборки, просто открой файл `config.json` в любом текстовом редакторе в папке `C:\tools\EVC\` и отредактируй его:
* В `ALLOWED_EXTENSIONS` заноси форматы, которые нужно считывать (например, `".py"`, `".js"`, `".html"`).
* В `IGNORE_DIRS` пиши папки, которые нужно пропускать (например, `"node_modules"`, `".git"`).
